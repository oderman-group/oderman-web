# Logos de Clientes

Coloca aquí los logos de tus clientes en formato PNG o JPG.

Nombres sugeridos:
- cliente-1.png
- cliente-2.png
- cliente-3.png
- cliente-4.png
- cliente-5.png
- cliente-6.png

**Recomendaciones:**
- Formato: PNG con fondo transparente o JPG
- Tamaño: Máximo 400px de ancho
- Resolución: 72-150 DPI
- Fondo: Preferiblemente transparente o blanco
